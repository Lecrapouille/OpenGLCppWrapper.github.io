glwrap::Transformable< T, n > Class Template Reference

a Transformable defines a 4x4 transformation matrix from a translation, a rotation and a scale. This class allow an object to move in a 2D-world (if n == 2U) or 3D-world (if n == 3U). Not a thread safe class. More...
