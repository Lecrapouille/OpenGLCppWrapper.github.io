◆ collinearity()

template<typename T , size_t n>
T glwrap::vector::collinearity ( Vector< T, n > const &  u,
Vector< T, n > const &  v 
)

Two non-null vectors are collinear if and only it exists a scalar k != 0, where: u = k v (note: if u is a null vector, any vector v is collinear to u because k = 0).

Returns
k if vectors are collinear, else return NaN if vectors are not collinear, else return 0 if zero vector.
Note
Use this function for T a familly of float and not for integers.
