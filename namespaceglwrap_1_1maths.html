template<typename T >
static T glwrap::maths::linspace ( T const  start,
T const  end,
size_t const  N,
std::vector< T > &  result,
const bool  endpoint 
)
static
Parameters
[in]startThe starting scalar value of the sequence.
[in]endThe end value of the sequence, unless endpoint is set to false. In that case, the sequence consists of all but the last of num + 1 evenly spaced samples, so that stop is excluded.
[in]NNumber of samples to generate. Should be > 1.
[out]Thevector of equally spaced samples in the closed interval [start, stop] or the half-open interval [start, stop) (depending on whether endpoint is true or false).
Returns
Size of spacing between samples. Return NaN if this value cannot be computed.
Note
: This code has been inpsired by the Numpy.linspace function.
