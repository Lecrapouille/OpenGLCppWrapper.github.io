Detailed Description

template<class T>
class glwrap::PendingContainer< T >

This class is used as buffer between CPU and GPU where pending elements are flushed to the GPU memory. Mostly method impacting on the container size have a side effect: they modify the estimator on the GPU memory usage for the current running application.
